# GitHub Pages setup

Use the `docs/` folder as your publishing source.

## Recommended setup

1. Push this `docs/` folder to your repository.
2. Open repository settings on GitHub.
3. Go to **Pages**.
4. Set the publishing source to your main branch and `/docs` folder.
5. Save.

## Optional custom domain

If you want a custom docs domain, configure it in GitHub Pages settings.

## Notes

- This docs set uses Jekyll with the `just-the-docs` remote theme.
- Edit `_config.yml` and replace `url` and `baseurl` with your real values.
- Replace the GitHub repository link in `_config.yml`.
