# Totistack Enterprise Feature Pack

Updated feature modules for the latest declarative Totistack structure.

## Included features

- forms
- integrations
- media
- search
- workflows

## Design goals

- declarative feature modules
- generated assembly compatibility
- one root shard-provider expected from Totistack core
- root auth/rbac compatibility
- production-oriented service factories
- starter Vue components that are generic and easy to extend
