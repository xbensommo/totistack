# Generated registries

This folder is generated by Totistack. Do not edit these files manually.

Generated files include:

- `server-actions.registry.js` — built from backend server-action declarations inside installed apps/features.
- `notifications.registry.js` — built from backend notification declarations inside installed apps/features.
- `functions.generated.js` — built from backend Firestore trigger declarations inside installed apps/features.

Project-specific backend declarations belong inside the app/feature that owns the business rule.
